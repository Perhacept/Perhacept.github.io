解决方案

手机端UI不适配

修改Canvas面板中的Canvas Scaler属性（没有就加组件）：

  • UI Scale Mode:Match Width Or Height
  • Reference Resolution(以P30为基准):2340x1080
  • Match:0.5

模型调整相关

  1. 轴心不在轴心（？）
    1. 建立个空物体，调整其坐标到想要的轴心位置
    2. 模型成为此空物体的子物体即可
